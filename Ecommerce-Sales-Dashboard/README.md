# 📊 Retail Sales Dashboard (Power BI)

## 🔍 Overview

This project is an interactive Power BI dashboard built to analyze retail sales data. It provides insights into revenue, customer behavior, and product performance.

---

## 🚀 Key Features

* KPI Cards: Total Revenue, Orders, Customers, AOV
* Revenue trend over time
* Top products by revenue
* Revenue by country
* Customer-wise revenue analysis
* Interactive slicers (Country, Date, Top N)

---

## 📊 Tools Used

* Power BI
* Data Modeling
* DAX (for measures like AOV, Top N)

---

## 📁 Dataset

Retail dataset containing:

* Invoice data
* Customer IDs
* Product descriptions
* Quantity and revenue

## 💡 Insights

* Majority of revenue comes from top countries
* Few products contribute to most sales (Pareto principle)
* Seasonal spikes visible in revenue trend

## 📌 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Use slicers to explore insights
