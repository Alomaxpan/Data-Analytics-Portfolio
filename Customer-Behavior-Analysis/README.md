# Customer Behavior Analytics

## Project Overview
This project analyzes customer behavior on an e-commerce platform
to identify purchasing patterns and business insights.

## Objectives
- Analyze customer spending patterns
- Understand membership-level behavior
- Evaluate the impact of discounts
- Identify top product categories

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Insights
- Platinum members generate the highest revenue.
- Discounts positively impact purchase amount.
- Mobile users contribute significantly to shopping activity.
- Customer satisfaction correlates with higher spending.

## Skills Demonstrated
- Exploratory Data Analysis
- Data Visualization
- Customer Segmentation
- Business Insight Generation
